# TDC

## This program is used for calculating:

- ### speed
- ### distance
- ### angle on bow
- ### visual length

## Requirements:

- ### HARDscope
- ### Realistic periscope magnification
- ### Uboat by Deep Water Studio

## All formulas are from [this](http://www.tvre.org/en/) website, mod descriptions, forums and mine algebra
